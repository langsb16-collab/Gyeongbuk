## 🚀 주요 기능

### 배달 서비스 (80%)
1. **전통시장 통합 장보기**
   - 반찬집 + 정육점 + 과일가게 1회 주문
   - 시장 묶음배달 시간대 운영
   - 장날 자동 알림

2. **로컬푸드 농가 직거래**
   - 당일 수확/당일 배송 표시
   - 생산자 실명·농가 위치
   - 지역화폐 추가 할인

3. **공공 추천 맛집**
   - 수수료 0%, 광고비 0원
   - 배달비 무료/최소화
   - 공정한 리뷰 시스템

### 중고·나눔 마켓 (20%)
1. **안전거래 존**
   - 경찰서/파출소 인근 장소
   - CCTV 밀집 지역 표시
   - 거래 예약/완료 체크

2. **중고거래**
   - 동일 시·군 내 거래
   - 시민 인증 (휴대폰+거주)
   - 분쟁 중재 시스템

3. **무료나눔**
   - 취약계층 연계
   - 재능·시간 나눔
   - 환경 보호 기여

### 경산 특산물·관광
1. **특산물 판매**
   - 대추 (지리적표시 등록)
   - 포도 (시설재배 거봉)
   - 복숭아 (천도복숭아 주산지)
   - 자두

2. **관광지 QR 커머스**
   - 팔공산 갓바위
   - 반곡지
   - 자인계정숲

3. **경산 대추축제**
   - 2025.10.17~10.19
   - 축제 연계 특가 판매
