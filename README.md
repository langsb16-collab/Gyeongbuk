# 경북 공공상생 플랫폼 - 프로토타입 데모

## 🎯 프로젝트 개요

**"배달 중심의 공공 플랫폼을 통해 소상공인·전통시장·로컬푸드의 실질적 매출을 지원하고, 중고·나눔 기능을 통해 시민 안전과 복지를 보완하는 경북형 생활 밀착 공공 디지털 인프라"**

- **사업명**: 경북 공공상생 플랫폼 (Gyeongsan Public Delivery & Share Market)
- **대상 지역**: 경산시 (시범) → 포항시, 구미시, 안동시 (확산)
- **서비스 비중**: 배달 80% / 중고·나눔 20%

## ✨ 핵심 기능

### 1. 배달 서비스 (80% - 메인)

#### 🏪 전통시장 통합 장보기
- **차별점**: 여러 점포를 1회 주문으로 묶음 배달
- 반찬집 + 정육점 + 과일가게 = 1회 결제
- 고령 상인 대행 등록 지원
- 시 보조 배달비 정책

#### 🌱 로컬푸드 농가 직거래
- 당일 수확 · 농가 직배송
- 생산자 실명 및 농가 위치 표시
- 친환경 인증 뱃지
- 지역화폐 추가 할인 혜택

#### 🍽️ 음식점 배달
- 중개수수료 0%
- 광고비 0원
- 배달비 무료/최소화
- 공공 추천 맛집 우선 노출

### 2. 중고·나눔 마켓 (20% - 보조)

#### 🛡️ CCTV·안전거래 존 (핵심 차별화)
- 경찰서, 파출소, 지구대
- 동 행정복지센터
- 공공 CCTV 밀집 지역
- 지도에서 안전거래 장소 자동 추천

#### ♻️ 중고거래 & 무료나눔
- 지역 기반 거래 (시·군 제한)
- 시민 인증 시스템
- 거래 후 평가 및 신뢰 점수
- 신고·중재 시스템

### 3. 지역화폐 & 쿠폰
- 경산 지역화폐 결제 연동
- 전통시장·로컬푸드 할인 쿠폰
- 첫 주문 무료배달 쿠폰
- 사용 내역 및 혜택 통계

### 4. 사전등록 가맹점 (오픈 예정)
- 200곳 가맹점 사전등록 진행 중
- "오픈 예정" 라벨로 앱 내 노출
- 오픈 알림 신청 기능
- 동의 기반 단계적 공개
