## 🎯 프로젝트 개요

**경북 광역 공공상생 플랫폼**은 경산시, 포항시, 구미시, 안동시를 대상으로 한 **수수료 0% 공공 배달 플랫폼 + 안전한 중고·나눔 마켓**을 통합한 시범 서비스입니다.

### 핵심 가치
- 🏪 **소상공인 지원**: 중개 수수료 0%, 광고비 0%
- 🌱 **지역 경제 활성화**: 로컬푸드 직거래, 전통시장 통합 장보기
- 🛡️ **안전한 거래**: CCTV·경찰서 인근 안전거래 장소 제공
- ♻️ **환경 보호**: 중고거래·무료나눔으로 재사용 촉진
- 💰 **지역화폐 연계**: 경산 지역화폐 결제 및 추가 적립

---

## 🏗️ 서비스 구조 (80:20 모델)

### 1. 배달 서비스 (80% 비중) - 메인 서비스

#### 📍 음식점 배달
- 공공 추천 맛집 자동 노출
- 위생우수·착한가격 업소 우선 배치
- 지역화폐 결제 시 추가 적립
- **배달비 무료** (시 보조)

#### 🏪 전통시장 통합 장보기
- 여러 가게 상품을 한 번에 주문
- 묶음 배달로 배달비 절감
- 장날 자동 알림
- 고령 상인 등록 대행 시스템

#### 🌾 로컬푸드 직거래
- 당일 수확 농산물 농가 직배송
- 생산자 실명·농장 소개
- 친환경 인증 표시
- 푸드 마일리지 감소 기여

### 2. 중고·나눔 마켓 (20% 비중) - 공공 안전 서비스

#### 🔐 안전거래 장소 (핵심 차별화)
- **CCTV 밀집 지역** 자동 추천
- **경찰서·파출소** 인근 거래
- **행정복지센터** 거래 공간 제공
- 안전거래 장소 지도 기능

#### ♻️ 중고거래 & 무료나눔
- 지역 제한 거래 (경산시 내)
- 거래 후 평가 시스템
- 신고·분쟁 행정 중재
- 재능·시간 나눔 기능

## 🚀 로컬 실행 방법

```bash
# 1. 의존성 설치
npm install

# 2. 개발 서버 실행 (Vite)
npm run dev

# 3. 빌드
npm run build

# 4. 프로덕션 미리보기
npm run preview

# 5. PM2로 실행 (샌드박스 환경)
npm run build
pm2 start ecosystem.config.cjs
```
