# 경북 공공상생 플랫폼 - 프로토타입 데모

## 🎯 프로젝트 개요

**경북 광역 공공상생 플랫폼**의 UI/UX 시연용 프로토타입입니다.

- **목적**: 시의회·행정 보고, 투자 유치, 개발 전 검증

- **특징**: 사전등록 가맹점 200곳 확보 시스템, CCTV·경찰서 인근 안전거래 기능

## 💡 핵심 기능

### 1. 배달 서비스 (80% 비중)

#### 📦 전통시장 통합 장보기
- **차별점**: 반찬집 + 정육점 + 과일가게를 1회 주문으로 묶음 배달
- **장점**: 고령 상인도 온라인 참여 가능, 시장 전체 매출 증가

#### 🌱 로컬푸드 농가 직거래
- **당일 수확**: 오늘 아침 수확한 신선 농산물
- **농가 직배송**: 생산자 실명, 농가 위치 표시
- **친환경 인증**: 유기농, 무농약, 친환경 뱃지

#### 🏆 공공 추천 맛집
- **수수료 0%**: 중개 수수료 없음
- **배달비 무료/최소화**: 시 보조금 지원
- **신뢰 점수**: 리뷰 조작 없는 공정한 평가

### 2. 중고·나눔 마켓 (20% 비중)

#### 🛡️ 안전거래 존 (핵심 차별화)
- **경찰서·파출소 인근** 거래 장소 추천
- **CCTV 밀집 지역** 자동 표시
- **동 행정복지센터** 거래 거점 제공
- **거래 예약 시스템** 시간 설정 및 완료 체크

#### ♻️ 중고거래 & 무료나눔
- **지역 제한**: 동일 시·군 내 거래만 허용
- **시민 인증**: 휴대폰 + 거주 확인
- **분쟁 중재**: 관리자 중재 구조

### 3. 가맹점 등록 시스템 (초간편 등록)

#### 📸 사진만 찍어도 등록 가능
- **사업자등록증 1장만**: OCR 자동 인식으로 3분 내 신청 완료
- **최소 입력 항목**: 상호명, 대표자명, 사업자번호, 주소, 연락처만 필수
- **메뉴 등록 나중에**: 신청 즉시 "무료배달 가맹점(준비중)" 표시
- **24시간 내 승인**: 관리자 검증 후 정식 가맹점 전환
- **고령자 친화**: 큰 글자, 간단한 화면, 현장 등록 모드 지원

#### 🔐 OCR 기반 온보딩
- **자동 인식**: 상호명, 대표자명, 사업자번호, 주소 자동 추출
- **신뢰도 평가**: 70% 이상 자동 통과, 낮으면 관리자 검증
- **실패 시 재촬영**: 촬영 가이드 제공
- **상태 관리**: APPLIED → PENDING_ACTIVE → ACTIVE 흐름

### 4. 관리자 검증 대시보드

#### 📊 실시간 통계
- **대기중 신청**: 승인 필요한 신청 건수
- **승인 완료**: ACTIVE 상태 가맹점
- **반려**: 서류 미비 등 반려 건수
- **전체**: 누적 신청 건수

#### ✅ 가맹점 심사
- **신청 목록**: 시·군별, 상태별 필터링
- **OCR 신뢰도**: 자동 인식 신뢰도 표시
- **승인/반려**: 원클릭 승인 또는 반려 사유 입력
- **서류 확인**: 사업자등록증 이미지 열람

#### 🎯 효율적 운영
- **24시간 내 처리**: 신청 후 1일 내 승인 목표
- **자동 검증 연계**: 국세청 API 연동 (실제 환경)
- **감사 로그**: 모든 승인/반려 이력 기록

### 주요 엔티티

1. **Restaurants** (음식점)
   - 카테고리, 평점, 배달시간, 배달비, 뱃지

2. **Market Products** (전통시장 상품)
   - 시장명, 점포명, 상품, 가격, 카테고리

3. **Local Foods** (로컬푸드)
   - 농가명, 수확일, 지역, 인증정보

   - 제목, 가격, 위치, 안전거래 여부

5. **Safe Zones** (안전거래 장소)
   - 타입 (경찰서/행정복지센터/CCTV)
   - 주소, 시설, 운영시간

6. **Leads** (사전등록 가맹점)
   - 상태 (LEAD_PRIVATE/INVITED/CONSENTED_PREVIEW/ACTIVE)
   - 동의 증빙, 상태 이력
