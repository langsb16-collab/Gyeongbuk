## 🔧 기술 세부사항

### 이미지 URL 형식
```
https://images.unsplash.com/photo-{photo-id}?w=300&h=200&fit=crop
```

### 파라미터 설명
- `w=300`: 너비 300px
- `h=200`: 높이 200px
- `fit=crop`: 비율 맞춤 자르기

### 이미지 라이센스
- **출처**: Unsplash (https://unsplash.com)
- **라이센스**: Unsplash License (무료 사용 가능)
- **상업적 이용**: 가능
- **저작권 표시**: 필수 아님 (권장)
