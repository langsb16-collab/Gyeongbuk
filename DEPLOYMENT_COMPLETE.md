## 🎨 주요 개선 사항

### 1. 완전 반응형 레이아웃
- **PC**: 3열 그리드, 큰 이미지 (200px), 넉넉한 폰트 크기
- **태블릿**: 2열 그리드 (sm 브레이크포인트)
- **모바일**: 1열 그리드, 컴팩트 레이아웃

### 2. 세련된 UI 요소
- **카드 호버 효과**: 상승 애니메이션 + 그림자 증가
- **부드러운 전환**: 0.2s transition
- **스티키 헤더**: 상단 고정, 뒤로가기 + 검색 버튼
- **눈에 띄는 배너**: "배달비 0원" 그라디언트 배경

### 3. 실전 데이터 연동
- API에서 음식점 목록 실시간 로딩 (`/api/restaurants`)
- 로딩 상태 표시 (스피너)
- 에러 처리 및 사용자 피드백
- 이미지 CORS 지원 + 오류 시 폴백

### 4. 주문 플로우 통합
- **메뉴 버튼**: `/store/{storeId}/menu` 이동
- **주문 버튼**: 
  1. POST `/api/cart/create` → cartId 생성
  2. localStorage 저장
  3. 메뉴 페이지로 리다이렉트
