# 경산온(ON) - 브랜드 아이덴티티

## 🎯 플랫폼명

**경산온(ON)**

### 의미
- **경산 + ON**: 경산이 켜져있다, 경산이 활성화되다
- **ON 중의적 의미**:
  - 온라인 플랫폼
  - 항상 켜져있는 서비스
  - 경산 전역 ON (전역 활성화)

## 💬 핵심 슬로건

### 메인 슬로건 (1순위)
**"경산은 배달비가 없습니다"**

### 서브 슬로건
**"배달비 0원, 경산의 기본"**

## ✨ 슬로건 장점

### 1. 즉각적인 메시지 전달
- "배달비 없음"을 3초 안에 이해
- 숫자 "0원"으로 강력한 인상
- 부정어 대신 긍정 프레임 ("없습니다" → 단호함)

### 2. 가맹점 모집 강점
- "우리 가게도 무료배달?" → 즉시 문의
- 경쟁 플랫폼 대비 명확한 차별점
- 상인 입장에서 이해하기 쉬움

### 3. 시민 혜택 강조
- "경산 시민"이라는 자부심
- 공공 플랫폼의 신뢰감
- 지역 정체성 강화

## 🌍 광역 확산 시 명칭 전략

### 경산 시범 (현재)
- **플랫폼명**: 경산온(ON)
- **슬로건**: "경산은 배달비가 없습니다"

### 포항/구미/안동 확산 시 (2가지 전략)

#### 전략 A: 지역별 독립 브랜드
- **포항온(ON)**: "포항은 배달비가 없습니다"
- **구미온(ON)**: "구미는 배달비가 없습니다"
- **안동온(ON)**: "안동은 배달비가 없습니다"

#### 전략 B: 통합 브랜드
- **플랫폼명**: 경북온(ON) 또는 경북제로배달
- **슬로건**: "경북은 배달비가 없습니다"

## ⚠️ 주의사항

### 광역 확산 시 명칭 조정 필요
- 경산→포항/구미/안동 확산 시 지역명 변경
- 통합 브랜드 vs 지역별 브랜드 전략 선택
- 슬로건 일관성 유지 ("○○은 배달비가 없습니다")

### 법적 검토 사항
- "무료"/"0원" 표현 공정거래법 준수
- 실제 배달비 정책과 100% 일치 확인
- 예외 조건 명시 (거리 제한 등)
